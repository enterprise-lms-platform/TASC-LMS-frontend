# Course Thumbnail Images

This folder contains category-based thumbnail images for course cards.

## Current Status

**Available Images** (✅ Real JPEGs):
1. **safety.jpg** - EHS / Safety courses (4032x2268, real photo)
2. **welding.jpg** - Welding courses (3000x2400, real photo)

**Missing Images** (⚠️ Need to be added):
3. **electrical.jpg** - Electrical courses (circuits, wiring)
4. **plumbing.jpg** - Plumbing courses (pipes, tools)
5. **scaffolding.jpg** - Scaffolding courses (construction scaffolding)
6. **business.jpg** - Business/Assessment courses (office, documents)
7. **certification.jpg** - Certification courses (diploma, achievement)

**Note**: Categories without specific images temporarily use `safety.jpg` or `welding.jpg` as fallback.

## Required Format

All images must be:
- **Format**: JPEG (not SVG, PNG, or other formats)
- **Dimensions**: 800x400px (2:1 aspect ratio)
- **File size**: <100KB each (optimized)
- **Quality**: High resolution, professional photos

## Sources

Free stock photos can be downloaded from:
- Unsplash: https://unsplash.com
- Pexels: https://www.pexels.com
